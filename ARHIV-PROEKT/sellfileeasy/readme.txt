Описание продукта:

  Для использования данного продукта необходим акаунт в системе ROBOKASSA.
  Плагин позволяет ограничивать доступ к файлам. Оплата
  доступа к файлу производится путем перечисления пользователем указанной суммы
  владельцу сайта, через платежную систему ROBOKASSA.

Принцип работы:

  После установки и настройки плагина, владелец сайта сможет указывать ссылки
  на файлы ниже приведенным способом, для ограничения к ним доступа. После
  нажатия на одну из таких ссылок, скачивание файла не начнется до тех пор пока
  пользователь не произведет оплату. Ссылка на оплату высылается
  на указанный пользователем при покупке email.

Установка:

  1. Скопируйте папку плагина sellfileeasy в корень сайта
  2. Сконфигурируйте файл sellfileeasy/config.php следуя комментариям.
  3. Скопируйте файлы вашего архива в папку
      /sellfileeasy/archive/цена файлов лежащих в этой папке/
  4. Дайте права 666 файлу sellfileeasy/archive/transaction.log
  5. Ссылка на файл для скачивания должна выглядеть так:
    <a href = 'http://ваш домаин.ру/sellfileeasy/archive/199/файл для скачивания.zip'>скачать картинку</a>

  6.a. Сконфигурируйте файл sellfileeasy/robokassa/config.php следуя комментариям.
  На сайте ROBOKASSA указываем:
    *Result URL:- http://ваш сайт.ru/sellfileeasy/robokassa/result.php
    *Success URL:- http://ваш сайт.ru/sellfileeasy/robokassa/success.php
    *Fail URL:- http://ваш сайт.ru/sellfileeasy/robokassa/fail.php

  6.b. Сконфигурируйте файл sellfileeasy/z-payment/config.php следуя комментариям.
  На сайте Z-payment.com указываем:
    *Result URL:- http://ваш сайт.ru/sellfileeasy/z-payment/result.php
    *Success URL:- http://ваш сайт.ru/sellfileeasy/z-payment/success.php
    *Fail URL:- http://ваш сайт.ru/sellfileeasy/z-payment/fail.php

  6.c. Сконфигурируйте файл sellfileeasy/interkassa/config.php следуя комментариям.
  На сайте Interkassa указываем:
    *Success URL:- http://ваш сайт.ru/sellfileeasy/interkassa/success.php
    *Fail URL:- http://ваш сайт.ru/sellfileeasy/interkassa/fail.php
    *URL проверки отключите
    *Result URL:- http://ваш сайт.ru/sellfileeasy/interkassa/result.php


  Статистика: http://ваш домаин.ру/sellfileeasy/stat.php
  Пароль и логин указаны в файле sellfileeasy/config.php
     //Statistics Login
     $stat_login = "";
     //Statistics Password
     $stat_pass = "";

    Установка завершена, хорошего заработка!
